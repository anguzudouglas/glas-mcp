API Reference

Plug & play from anywhere.

Every endpoint returns the Glas Unified Response Schema (GURS). All responses include ok, result, error, and meta.

Base URL (Production)
https://glas-mcp.onrender.com
Response Schema (GURS)
{ ok, tool, result, error, meta }

Quick Start — JavaScript

const BASE = "https://glas-mcp.onrender.com/api/v1";

async function callTool(name, args) {
  const res = await fetch(`${BASE}/tools/${name}`, {
    method: "POST",
    headers: { "Content-Type": "application/json" },
    body: JSON.stringify({ arguments: args }),
  });
  const { ok, result, error } = await res.json();
  if (!ok) throw new Error(error.message);
  return result;
}

// Web search
const r = await callTool("web_search", { query: "MCP tools", num_results: 5 });

// Generate a chart → get base64 PNG
const chart = await callTool("plot_chart", {
  chart_type: "bar",
  data: { categories:["Q1","Q2","Q3"], values:[[100,145,130]], labels:["Revenue"] },
  title: "Quarterly Revenue", colors: ["#00c8f8"], style: "ggplot"
});
img.src = `data:image/png;base64,${chart.image_base64}`;

About

Built by Anguzudouglas.

Glas MCP is an open-source, modular Model Context Protocol server. Tools are auto-discovered from subdirectories — add a folder with main.py + tool.yaml and it's live on next restart. Skills guide the agent to produce better outputs from each tool.

Quick deploy with Docker
git clone https://github.com/anguzudouglas/glas_mcp
cd glas_mcp
cp .env.example .env

# Single container
docker build -t glas-mcp .
docker run -p 8000:8000 glas-mcp

# Or with compose
docker-compose up
